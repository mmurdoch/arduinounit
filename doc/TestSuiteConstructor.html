ArduinoUnit Documentation

ArduinoUnit Library Reference

TestSuite()

Description

Creates a variable of type TestSuite.

Syntax

TestSuite()
TestSuite(name)

Parameters

name: the name of the test suite (optional, defaults to empty string)

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { Serial.begin(9600); } void loop() { suite.run(); }
