ArduinoUnit Library Reference

getReporter()

Description

Returns the reporter for a test suite.

Syntax

suite.getReporter()

Parameters

suite: the test suite for which to set the reporter

Returns

the reporter for the specified test suite

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { Serial.begin(9600); Reporter& reporter = suite.getReporter(); } void loop() { suite.run(); }
