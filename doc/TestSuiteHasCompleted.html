ArduinoUnit Library Reference

hasCompleted()

Description

Returns whether or not a test suite has completed execution.

Syntax

suite.hasCompleted()

Parameters

suite: the test suite to query

Returns

true if the test suite has completed execution, false otherwise

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { Serial.begin(9600); } void loop() { // completed == false bool completed = suite.hasCompleted(); suite.run(); // completed == true completed = suite.hasCompleted(); }
