ArduinoUnit Library Reference

setReporter()

Description

Sets a test suite reporter.

Syntax

suite.setReporter(reporter)

Parameters

suite: the test suite for which to set the reporter
reporter: the reporter to set

Example

#include <ArduinoUnit.h> TestSuite suite; LcdReporter reporter; void setup() { suite.setReporter(reporter); } void loop() { suite.run(); }
