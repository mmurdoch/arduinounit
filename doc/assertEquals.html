ArduinoUnit Documentation

ArduinoUnit Library Reference

assertEquals()

Description

Asserts that an integer value is equal to another integer value.

Syntax

assertEquals(expected, actual)

Parameters

expected: expected value of the integer
actual: integer value to check against the expected value

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { Serial.begin(9600); } test(constrain) { int upper = 40; int lower = 30; int toConstrain = 20; int constrained = constrain(toConstrain, lower, upper); assertEquals(lower, constrained); } void loop() { suite.run(); }
