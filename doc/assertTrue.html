ArduinoUnit Library Reference

assertTrue()

Description

Asserts that a value is true.

Syntax

assertTrue(value)

Parameters

value: boolean value to check

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { Serial.begin(9600); } test(comparison) { assertTrue(7 > 6); } void loop() { suite.run(); }
